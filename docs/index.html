Formati di visualizzazione

Scegli come leggere la fattura

Ogni formato è pensato per uno scopo diverso: lettura quotidiana, conformità ministeriale o debug tecnico.

Elegante
Formato BrioFEview

Rappresentazione di cortesia costruita dall'app stessa: decodifica i codici della fattura (modalità di pagamento, regime fiscale, natura IVA...) in etichette leggibili, con un'impaginazione pulita pensata per la lettura veloce. È il formato consigliato per l'uso quotidiano.

Ministeriale

Il foglio di stile ufficiale dell'Agenzia delle Entrate, identico a quello usato dal portale Fatture e Corrispettivi.

AssoSoftware

Il foglio di stile alternativo diffuso tra i software gestionali italiani, per confrontare la resa con altri programmi.

Codice XML

Il contenuto XML grezzo, riformattato ed evidenziato con numeri di riga: utile per verificare rapidamente un dato o per il debug di una fattura anomala.

Guida rapida

Come si usa

Apri un file o una cartella

Dal pulsante in toolbar, con "Apri con" da Esplora Risorse, oppure trascinando il file o la cartella nella finestra.

Scegli il formato

Dal menu "Vista" seleziona Elegante, Ministeriale, AssoSoftware o Codice XML.

Stampa o salva il PDF

Un clic sui pulsanti Stampa o Salva PDF nella toolbar: nessuna configurazione aggiuntiva richiesta.

Esporta un'intera cartella

Con la barra laterale aperta, seleziona le fatture da includere e usa "Esporta cartella" per PDF multipli o unificati.

Versione portable

Un unico file eseguibile, senza installazione: comoda da una chiavetta USB o per una prova rapida senza toccare il sistema.

Scarica il portable

In alternativa, BrioFEview si può avviare direttamente dai sorgenti (richiede Python 3.11+):

Windows, dai sorgenti

Avvio con doppio clic su Avvia BrioFEview.bat, oppure da terminale:

pip install -r requirements.txt python -m visualizzatore

macOS, dai sorgenti

Al primo avvio lo script crea automaticamente un ambiente virtuale e installa le dipendenze:

chmod +x avvia_briofeview.command ./avvia_briofeview.command

Il codice sorgente completo, le istruzioni dettagliate e le note di rilascio sono disponibili nel repository GitHub.
